This guide will help you install HL2:SB++ on Android devices.

Requirements

  • Android device
  • Source Engine 1.17 APK
  • Half-Life 2 files or Source Universal Patch

PC builds are technically supported but not recommended, since Garry's Mod already exists on PC.

Step 1 - Install Source Engine

Download and install Source Engine 1.17 APK.

Download Source Engine 1.17

Please support the original developers afterwards.

Source Engine Discord Server

Step 2 - Prepare Half-Life 2 Files

If you do not own Half-Life 2, download Source Universal Patch.

Download Source Universal Patch

If you already own Half-Life 2, use the pre-20th anniversary files.

Extract the following folders into your srceng/ directory: 

hl2/
platform/

Step 3 - Install HL2:SB++

  1. Download the HL2:SB++ APK.
  2. Install the APK.
  3. Download the resource package.
  4. Extract the resources into srceng/.
  5. Make sure the folder is named exactly:
hl2sbpp

Step 4 - Launch the Game

Open the installed HL2:SB++ application and start playing.

Additional Game Content

Content from games such as Counter-Strike: Source and Portal must be downloaded separately.
